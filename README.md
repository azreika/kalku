# kalku

Simple calculator written in Rust.

Evaluates arbitrary arithmetic expressions involving addition, subtraction, or multiplication.

## Usage

Using cargo, just run:

`cargo run`

## Examples

```
> 1 + 2
3
```

```
> 1 + 2 * (3 - 2)*1  + 1 * (22-(3)+(4))
26
```

```
> -1 - (1+2) - - 2
-2
```
